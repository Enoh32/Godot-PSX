# Godot-PSX
My own version of PSX-style shaders for godot. Originally derrived from marmitoTH's, but modified quite heavily.

Includes:
PSX standard,
PSX alpha-cut,
PSX water

Features:
Corrected normals,
FOV-corrected vertex snapping,
Near and Far triangle culling,
Affine texture-mapping,
Vertex Lighting
